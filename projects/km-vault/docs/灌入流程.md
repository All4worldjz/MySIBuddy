# raw/ 文档灌入流程（灌入即编译）

## 说明
此脚本用于将飞书文档快速导入 raw/ 目录，并触发 LLM 编译。

## 使用方法

### 方法一：通过飞书文档 URL（推荐）
```bash
bash /home/admin/.openclaw/scripts/feishu_import_to_raw.sh \
  --url "https://pbrhmf5bin.feishu.cn/docx/UpSAdrgZ7oHGDTxNnwXcCkJbn9c" \
  --vault_token "QB50fa4HYlYPCRd5Q8Cck6MMnvf"
```

### 方法二：通过飞书文档 token
```bash
bash /home/admin/.openclaw/scripts/feishu_import_to_raw.sh \
  --token "UpSAdrgZ7oHGDTxNnwXcCkJbn9c" \
  --vault_token "QB50fa4HYlYPCRd5Q8Cck6MMnvf" \
  --title "Karpathy LLM Wiki"
```

## 脚本工作流程

1. **获取文档内容**：`feishu_fetch_doc` → Markdown
2. **清理标题**：移除 "刚刚，Karpathy 开源个人 LLM Wiki" 等标题
3. **保存到 raw/**：`raw/yyyymmdd_HHMM_title.md`
4. **记录元数据**：`raw/.metadata.json`（来源 URL、时间戳）
5. **触发编译**：`sessions_spawn({agentId: "coder-hub", task: "编译 raw/xxx.md"})`

## cron 配置（自动扫描）

编辑 `/etc/crontab`：
```bash
# 每 5 分钟扫描一次 raw/，发现新文件立即编译
*/5 * * * * admin bash /home/admin/.openclaw/scripts/compile_raw.sh
```

## 编译触发脚本（compile_raw.sh）

```bash
#!/bin/bash
# 灌入即编译触发器

VAULT_RAW_DIR="/home/admin/.openclaw/workspace-tech-mentor/KM_Vault/raw"

# 检测新文件
for file in $VAULT_RAW_DIR/*.md; do
  if [[ -f "$file" ]]; then
    filename=$(basename "$file")
    
    # 检查是否已处理（元数据检查）
    if ! grep -q "$filename" "$VAULT_RAW_DIR/.metadata.json" 2>/dev/null; then
      echo "[$(date)] 发现新文件: $filename，触发编译..."
      
      # 调用 coder-hub 组件编译
      sessions_spawn({
        "agentId": "coder-hub",
        "message": "根据 schema/AGENTS.md 和 schema/WIKI_SCHEMA.md 规则，将 raw/$filename 编译为 wiki 实体页/概念页",
        "model": "minimax/MiniMax-M2.7"
      })
      
      # 记录元数据
      echo "{\"file\": \"$filename\", \"imported_at\": \"$(date -Iseconds)\"}" >> "$VAULT_RAW_DIR/.metadata.json"
    fi
  fi
done
```

## 支持的文件格式

| 格式 | 来源 | 处理方式 |
|------|------|---------|
| `.md` | 飞书文档/网页剪存 | 直接使用 |
| `.docx` | 飞书文档导出 | 解析为 Markdown |
| `.pdf` | PDF 文件 | OCR → Markdown（OCR 后处理） |

## 命名规范

**文件名格式**：`yyyymmdd_HHMM_title.md`

**示例**：
- `20260408_0730_Karpathy_LLM_Wiki.md`
- `20260408_0735_OpenClaw_Architecture.md`

## 元数据字段（.metadata.json）

```json
[
  {
    "file": "20260408_0730_Karpathy_LLM_Wiki.md",
    "imported_at": "2026-04-08T07:30:00+08:00",
    "source_url": "https://pbrhmf5bin.feishu.cn/docx/...",
    "title": "Karpathy LLM Wiki",
    "size": 13486,
    "processed": false
  }
]
```

## 错误处理

| 错误 | 处理方式 |
|------|---------|
| 飞书文档不存在 | 记录日志，跳过 |
| Markdown 解析失败 | 保存原文件到 `raw/errors/` |
| LLM 编译失败 | 记录到 `wiki/reports/errors_YYYY-MM-DD.log` |

## 权限要求

- **技术权限**：飞书 `feishu_fetch_doc`, `sessions_spawn`
- **写入权限**：`raw/` 目录 rw
- **调用 Agent**：coder-hub（minimax/MiniMax-M2.7）
