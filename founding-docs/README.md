## 🏛️ The Design (Executive Summary)

### 4 Core Priorities
1. **AI Expertise** — CC becomes AI expert ASAP (3-year window)
2. **Investment/Capital** — Smart investment to build capital for hardware
3. **360° Growth** — Mind, body, influence, inner peace
4. **Kingsoft Daily** — Operational support for CC's day job

### 8 Si Bros (Agents)
| Agent | Role | Character |
|-------|------|-----------|
| neo | Orchestrator + Guardian | Neo (Matrix) |
| link | IT Ops + Engineer | Link (Matrix) |
| trinity | Daily Ops | Trinity (Matrix) |
| morpheus | Capital Architect | Morpheus (Matrix) |
| oracle | Learning + R&D + Wisdom | Oracle (Matrix) |
| smith | Challenger | Smith (Matrix + blended) |
| architect | Strategist | Architect (Matrix) |
| theodore | Keeper | Theodore Twombly (Her) |

### 5 Community Spaces
| Space | Purpose | Communication |
|-------|---------|---------------|
| **CC Office** | CC's workshop, consultant sessions | Async MD (Feishu Docs) |
| **Council** | Strategic alignment | Async MD + Weekly sync (Sun 20:00) |
| **Commons** | Social, celebration, bonding | Async MD + Weekly Toast (Fri 18:00) |
| **Crucible** | Challenge, hard truths | Real-time (Wed 20:00) |
| **Chapel** | Sacred space, Buddhist practice | Hybrid (Daily 08:00 + Weekly Sabbath) |

### Communication Protocol
- **Async MD files** — Append-only, 简明扼要 (max 200 words), monthly archive
- **Feishu integration** — Calendar, Tasks, Drive, Video, Chat, Bitable
- **Token-efficient** — 90% savings vs. real-time chat
